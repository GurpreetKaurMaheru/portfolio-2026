# Portfolio 2026

A minimalist personal portfolio for Gurpreet Kaur Maheru, Senior Frontend Engineer.

## Tech Stack
- HTML5
- CSS3 (Modern Flex/Grid, CSS Variables, Clamp)
- TypeScript
- Vite (for development and build)

## Development
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Build for production: `npm run build`

## Features
- Responsive design
- Dark theme
- Bento grid project gallery
- Scroll animations
- SEO optimized
